Aplikace ÚKOLY - PouchDB API

Aplikace úkoly je ukázkou použití offline databáze PouchDB běžící přímo v prohlížeči, která umožňuje ukládat data lokálně. Tato API je psaná v JavaScriptu a běží pod systémem Node.js. Umožňuje univerzální použití a je open source. Aplikace se skládá ze 2 elementů - hlavičky a obsahu. Hlavička obsahuje pole pro vložení textu a v obsahu se vypisují zadané úkoly. Práce s API probíhá standartně voláním metod PouchDB: db.doSomething(args..., [options], [callback]){}. Používají se i standartní callbacky Node.js: function(error, result) {}.. Kompletní dokumentace je na oficiálním webu PouchDB.

Zdrojový kód JavaScriptu je k dispozici v repozitáři GitHub.

    Google Maps API

    Aplikace Google Maps API je dnes již standartní API, která se využívá k mapovým službám. Díky spoustě již hotových funkcí je aplikace snadno použitelná pro téměř jakékoliv použití a obsahuje množství nastavení a stylování map.

    Její použití je opravdu jednoduché - nejprve vytvoříme proměnnou map, do které následně funkcí initMap přiřadíme objekt Map dle výběru: var map; function initMap() { map = new google.maps.Map();.


    Kontaktní formulář

    Kontaktní formulář je ukázkou integrace s Netlify Forms API, které umožňuje obsluhovat data z formulářů. Koncový bod Netlify Forms API je integrovaný přímo v rámci hostingu Netlify a sbírá data prostřednictvím HTTP (metodami GET a POST).
