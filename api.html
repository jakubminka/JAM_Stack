Aplikace ÚKOLY

Aplikace úkoly je ukázkou použití offline databáze PouchDB běžící přímo v prohlížeči, která umožňuje ukládat data lokálně. Tato API je psaná v JavaScriptu a běží pod systémem Node.js. Umožňuje univerzální použití a je open source.
