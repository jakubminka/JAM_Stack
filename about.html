Co znamená JAM Stack?

JAM Stack je pojmenování moderní technologie pro vývoj webu a webových aplikací založené na klientském JavaScriptu, API a Markupu.

JavaScript

O veškterou dynamickou funkčnost webu se stará JavaScript a to kompletně na straně klienta - ať už ve formě frontendového frameworku, knihovny (Vue.js, Angular, React) nebo tzv. vanilla JavaScriptu.

API

API neboli Application Programming Interface neboli česky rozhraní pro programování aplikací je sada rutin, protokolů a nástrojů pro vytváření softwarových aplikací. V zásadě rozhraní API určuje, jak by měly softwarové komponenty interagovat.

Markup

Pod pojmem Markup neboli značkovací jazyk se skrývá veškterý textový obsah webu. Ať už jde o klasické HTML nebo CSS, ale v JAM Stacku je možné použít i další značkovací jazyky. Nejpoužívanějšími jsou Markdown, AsciiDoc nebo Textile.
